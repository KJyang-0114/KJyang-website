# KJyang的小小網站

這是一個使用 Vue 3 + Vite 構建的個人網站，主要用於展示遊戲精華時刻和個人介紹。

## 功能特點

- 🎮 遊戲精華時刻展示
  - 支援 Google Drive 視頻播放
  - 響應式視頻卡片布局
  - 全屏視頻播放體驗

- 👨‍💻 個人介紹頁面
  - 簡潔的自我介紹
  - 技能展示
  - 學習進度分享

- 📝 留言板功能
  - 即時留言發送
  - 留言歷史記錄
  - 簡潔的留言界面

- 🎨 現代化設計
  - 深色主題
  - 響應式布局
  - 流暢的動畫效果

## 技術棧

- 前端框架：Vue 3
- 構建工具：Vite
- UI 組件：Vuestic UI
- 圖標：Font Awesome
- 後端：Node.js + Express
